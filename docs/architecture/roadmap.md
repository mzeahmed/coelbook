# Roadmap

This roadmap describes the planned evolution of Coelbook.

It is intended as a guideline rather than a strict commitment. Priorities may change as the project evolves.

---

## v0.1.0 — Foundation

- [x] Initial project structure
- [x] React frontend
- [x] Go backend
- [x] PostgreSQL support
- [x] Installation wizard
- [x] Authentication
- [ ] Dashboard
- [ ] Book management
- [ ] Basic documentation pages

---

## v0.2.0 — Knowledge Base

- [ ] Create, edit and delete books
- [ ] Nested pages
- [ ] Markdown editor
- [ ] Table of contents
- [ ] Page history

---

## v0.3.0 — Search

- [ ] Full-text search
- [ ] Filters
- [ ] Tags
- [ ] Quick search (Command Palette)

---

## v0.4.0 — Attachments

- [ ] Image uploads
- [ ] File attachments
- [ ] Syntax highlighting
- [ ] Copy code blocks

---

## v0.5.0 — Collaboration

- [ ] User management
- [ ] Roles & permissions
- [ ] Shared books
- [ ] Activity log
